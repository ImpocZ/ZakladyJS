2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Remember to use camel case! (for instance when a function is composed of more words - 'printRealNumbers')
  2. Never start a name of a variable with a number, @, (), %, +, #, "" etc.. Though it can start with a $ or _.
  3. Remember, to be careful when writing your code! JavaScript is case sensitive! - "Console.Log("Hello world.")" will not work like this.
  4. Do not Confuse JavaScript with Java, you will just look like a complete bafoon.
  5. Also keep your code clean, if you will do spaggeti code, then everyone will laught at you. + comments are done via // - one line and /**/ - multiple lines. (.js)

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, %, (modulo - zbytek po celočíselném dělení), ** (umocnění), ++, --

Přiřazovací

=, +=, -=, /=, *=, **=, %= (rovná se zbytku po celočíselném dělení)

Logické

&& (AND), || (OR), ! (NOT);

Porovnávací

==, === (kontroluje nejen hodnotu, ale i typ proměnné.), !=, !== (nerovná se to hodnotě, nebo proměnné), >, <, >=, <=, ? (ternární operátor);

Bitové

&, |, ~ (not), ^ (xor), << (levý bitshift), >> (pravý bitshift), >>> (unsigned pravý bitshift);

Typové

typeof (typ proměnné), instanceof (vratí true, jestli je objekt instancí type (object));

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* vyskočí pop-up alert okno s varováním */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt("Zadej heslo.");

confirm()

/* Vyskočí okno, ve kterém můžeme vybrat 'OK' nebo 'CANCEL', podle našeho výběru nás stránka přesměruje, či ne. */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování: pop-up okno narušuje průběh procesů a může omezovat uživatele.
